## Comment utiliser cette documentation

1. **Pour lever des fonds maintenant** → lire 01, puis envoyer 05 + 06, garder 02/03/04 pour les questions.
2. **Pour signer un partenaire** → 07 puis le bon modèle dans 08.
3. **Pour rassurer un investisseur technique** → 09, 10, 11.
4. **Pour décider quoi construire ensuite** → 12 puis 13 pour le budget.

> ⚠️ **Avertissement juridique & financier :** les modèles de contrats, les statuts et les chiffres financiers de cette documentation sont des **bases de travail professionnelles**, PAS des conseils juridiques ou comptables validés. Avant signature ou dépôt officiel, faites relire par un **avocat d'affaires** et un **expert-comptable** habilités dans votre juridiction (Togo / OHADA).
